Crawling Berita#

Crawling dilakukan untuk mendapatkan berita berita dari sumber yang digunakan. Pada crawling ini digunakanlah website beritasatu.com dan mengambil kategori Politik, Olahraga dan Teknologi

Install library#

Library yang digunakan untuk mengambil berita dari website,dan juga pemrosesan antara lain:

  • requests : Untuk meminta permintaan HTTP ke server

  • BeautifulSoup : Untuk mengekstrak data dari dokumen HTML dan XML

  • csv : Untuk menyimpan data dalam format csv

  • pandas : Untuk menampilkan data csv

import requests
from bs4 import BeautifulSoup
import csv
import pandas as pd

Proses Crawling#

Pemilihan kategori berita berdasarkan input yang diinginkan oleh pengguna ingin mengambil berita tentang apa. Pada crawling kali ini berita yang diambil ialah tentang Politik, Olahraga dan juga Teknologi.

# Daftar kategori yang dapat diisi custom sesuai kebutuhan
kategori = ['Politik', 'Olahraga', 'Teknologi']

# Jumlah data yang diambil per kategori
jumlah_data_per_kategori = 350

# List untuk menyimpan data
data_list = []

# Looping untuk setiap kategori
for idx, kategori_terpilih in enumerate(kategori, start=1):
    katakunci = kategori_terpilih

    # Looping untuk setiap halaman
    for halaman in range(1, jumlah_data_per_kategori // 20 + 1):  # 20 berita per halaman
        url = "https://www.beritasatu.com/search/{}/{}".format(katakunci, halaman)
        headers = {
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:109.0) Gecko/20100101 Firefox/119.0'
        }
        req = requests.get(url, headers=headers)
        soup = BeautifulSoup(req.text, 'html.parser')
        items = soup.findAll('div', 'row mt-4 position-relative')

        # Looping untuk setiap berita dalam halaman tertentu
        for it in items:
            # Looping judul
            judul = it.find('h2', 'h5 fw-bold').text.strip()
            # Looping tanggal berita
            tanggal_tag = it.find('span', 'b1-date text-muted')
            tanggal = ''
            if tanggal_tag:
                tanggal = ' '.join(tanggal_tag.text.strip().split('\n'))
            # Looping link berita untuk diambil data kontennya
            link = it.find('a', 'stretched-link')
            if link:
                href = link.get('href')
                response = requests.get(href, headers=headers)
                berita_soup = BeautifulSoup(response.text, 'html.parser')
                #Looping untuk konten berita jika link bisa diakses
                konten_tag = berita_soup.find('div', 'col b1-article body-content')
                konten = ''
                if konten_tag:
                    konten = konten_tag.text.strip()

                # Menambahkan data ke dalam list
                data_list.append({ 'Judul': judul, 'Tanggal': tanggal, 'Link': href, 'Isi': konten, 'Label': katakunci,})

        # Menampilkan progres
        progress = (halaman - 1) * 20 + len(items)
        total_data = idx * jumlah_data_per_kategori
        print(f"{progress}/{total_data} data terambil untuk kategori {kategori_terpilih}")

# Membuat dataframe dari list data
df = pd.DataFrame(data_list)
print("Pengambilan data selesai.")

Normalisasi#

Proses normalisasi atau preprocessing data dilakukan agar mendapat data yang berkualitas

Import library#

Dalam proses normalisasi ada beberapa library yang digunakan :

  • nltk: Library untuk pemrosesan bahasa alami (Natural Language Processing).

  • word_tokenize digunakan untuk memecah teks menjadi token (kata-kata).

  • stopwords adalah daftar kata umum yang seringkali tidak memberikan nilai tambah dalam analisis teks, dan umumnya dihapus untuk fokus pada kata-kata kunci.

  • sklearn Library untuk machine learning dan data mining.

  • TfidfVectorizer digunakan untuk mengonversi kumpulan dokumen teks menjadi representasi vektor TF-IDF.

  • CountVectorizer digunakan untuk mengonversi kumpulan dokumen teks menjadi representasi vektor berdasarkan hitungan frekuensi kata.

  • numpy: Library untuk operasi matematika dan array.

  • re: Modul ekspresi reguler untuk manipulasi string.

  • warnings.filterwarnings('ignore') untuk mengabaikan atau menyembunyikan peringatan (warnings) yang mungkin muncul selama eksekusi kode. Dalam konteks ini, peringatan yang muncul akan diabaikan.

  • nltk.download('stopwords') dan nltk.download('punkt') fungsinya untuk mengunduh dataset stopwords dan model tokenisasi punkt dari NLTK jika belum ada di sistem.

all_results = []

for num_topics in k:
    for alpha_val in alpha:
        for beta_val in beta:
            # Inisialisasi model LDA dengan parameter tertentu
            lda = LatentDirichletAllocation(n_components=num_topics, doc_topic_prior=alpha_val, topic_word_prior=beta_val)

            # Latih model LDA pada training set yang telah divektorisasi
            lda_train_data = lda.fit_transform(X_train)
            lda_test_data = lda.transform(X_test)

            # Inisialisasi model klasifikasi SVM
            current_classifier_model = SVC()

            # Latih model klasifikasi pada data pelatihan
            current_classifier_model.fit(lda_train_data, y_train)

            # Lakukan prediksi pada data pengujian
            y_pred_test = current_classifier_model.predict(lda_test_data)

            # Hitung akurasi
            current_accuracy = accuracy_score(y_test, y_pred_test)

            print(f"Jumlah Topik: {num_topics}, Alpha: {alpha_val}, Beta: {beta_val}, Accuracy: {current_accuracy}")

            # Simpan hasil
            all_results.append({
                'Model': 'SVM',
                'Jumlah Topik': num_topics,
                'Alpha': alpha_val,
                'Beta': beta_val,
                'Accuracy': current_accuracy
            })

            # Cek akurasi tertinggi untuk best_results
            if current_accuracy > best_results['accuracy']:
                best_results['k'] = num_topics
                best_results['alpha'] = alpha_val
                best_results['beta'] = beta_val
                best_results['accuracy'] = current_accuracy
                best_results['lda_test'] = lda_test_data
                best_results['lda_train'] = lda_train_data

                best_classifier_model['classifier_model'] = current_classifier_model
